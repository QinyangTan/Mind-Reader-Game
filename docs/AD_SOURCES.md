# Ad Media Sources

Mind Reader keeps its logo, mascot, and UI art original to this repo. The only external media added for the sponsor placeholders in this content expansion pass are the two scenic ad assets below.

## Imported ad media

| Local file | Used in | Source | License note |
| --- | --- | --- | --- |
| `public/media/ads/night-railroad-station-1440.jpg` | Poster-style leaderboard, rail, and mobile sponsor examples | [Wikimedia Commons: Nighttime Railroad Station (Unsplash)](https://commons.wikimedia.org/wiki/File:Nighttime_Railroad_Station_(Unsplash).jpg) | Commons lists this file as CC0 / public domain dedication from the pre-2017 Unsplash license. The repo stores a resized derivative for web use. |
| `public/media/ads/photoreal-train-360p.webm` | Optional video sponsor example in the right rail | [Wikimedia Commons: Photoreal-train.webm](https://commons.wikimedia.org/wiki/File:Photoreal-train.webm) | Commons marks the file public domain. The repo uses the 360p WebM transcode for a lighter in-app preview. |

## Notes

- These files are used only as example sponsor media inside `AdSlot` / `SponsorRail` placeholders.
- All ad copy, layout, styling, and fallback house creatives remain original to this repo.
- No third-party ad network code, trackers, or embeds are included by default.
